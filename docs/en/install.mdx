# Install

Use the following steps to install the component.

## Downloading

:::info
To use the hosted control plane, you need two cluster plugins from Customer Portal.
1. **Alauda Container Platform Provider Kubeadm**
2. **Alauda Container Platform Hosted Control Plane**
3. **Alauda Container Platform SSH Infrastructure Provider**
:::

:::info
To deploy etcd operator, you need to download the app named **Alauda Build Of etcd Operator** from Customer Portal.
:::
